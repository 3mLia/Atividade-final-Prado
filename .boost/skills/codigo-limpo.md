# Skill: Código Limpo e Padrões Laravel

## Diretrizes de Escrita de Código

1. **Nomenclatura:** PascalCase para classes, camelCase para variáveis/funções. Tabelas e URLs no plural em inglês.
2. **Early Return:** Evite aninhamentos profundos de `if` usando retornos antecipados.
3. **Língua:** Código (variáveis, métodos, models) em Inglês. Textos para o usuário (views, botões, alertas) em Português do Brasil.

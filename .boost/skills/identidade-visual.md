# Skill: Identidade Visual e Layout (Barbearia)

## Diretrizes de Design e UI/UX
Esta skill orienta a geração de views e componentes visuais para o sistema de agendamento de barbearia.

1. **Framework CSS:** Utilize exclusivamente o Tailwind CSS para toda a estilização.
2. **Paleta de Cores (Tema Barbearia):**
   - Fundo principal: Escuro/Dark (ex: `bg-slate-900` ou `bg-zinc-900`).
   - Fundo de painéis/cards: `bg-slate-800`.
   - Textos primários: Branco ou cinza bem claro (`text-gray-100`).
   - Cor de destaque (Acentos/Botões): Dourado/Ambar (ex: `text-amber-500`, `bg-amber-600`, `hover:bg-amber-700`).
3. **Tipografia:** Moderna e legível. Use classes Tailwind de tipografia para garantir hierarquia.
4. **Responsividade:** Todas as telas devem ser Mobile-First.
5. **Componentes:** Botões com bordas arredondadas (`rounded-md`) e transições suaves.
